Welcome to GitHub Pages.

Yet another sinatra boilerplate. This project is compatible with bower, and compiles assets files in realtime. I've been searching for best practice to handle assets in sinatra app. A lot of websites taught me that sprocket is the best answer. So far, you don't need grunt or other task runner. I use live reload so that manual rackup never bothers you. Of course, Sass, and Coffeescript are available. By default, I choose to use bootstrap, backbone. Enjoy hacking!
